---

ARM Demos is a collection of ROS 2 (Jazzy) applications for the **Colman** UR3e robotic arm platform. The suite demonstrates real-time hand gesture control via MediaPipe and OAK-D, vision-guided pick-and-place using AprilTag detection and a vacuum gripper, and eye-in-hand camera calibration workflows. All packages are managed with [Pixi](https://prefix.dev) for fully reproducible environments on Linux.
